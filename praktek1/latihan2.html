What is CSS?

CSS adalah bahasa untuk mengatur tampilan dokumen HTML.

What can I do with CSS?

CSS mengatur warna, ukuran, jarak, layout, dan banyak aspek visual.

Difference between CSS and HTML?

HTML menyusun struktur konten, CSS mengatur tampilannya.
